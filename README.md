# mochaFiTrader

## Build instructions (1st time)
1. create build directory
```bash
$ mkdir build
```

2. cd build/ directory
```bash
$ cd build/
```

3. cmake in build/ directory
```bash
$ cmake ..
```

4. make .exe file
```bash
$ make
```

5. execute .exe file (from project root)
```bash
$ ./build/mochaFiTrader
```
